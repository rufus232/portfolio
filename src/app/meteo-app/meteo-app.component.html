Description

Une application météo est un logiciel ou une application mobile conçue pour fournir des informations météorologiques en temps réel et des prévisions sur les conditions météorologiques. Ces applications permettent aux utilisateurs de consulter rapidement et facilement les prévisions météorologiques pour leur emplacement actuel ou pour d'autres régions. .

Ce projet utilise une base de donnée locale elle n'est pas encore mise en ligne ,je m'exuce pour la gêne occasionné
