Description

Un jeu de quiz est un type de jeu dans lequel les joueurs sont mis au défi de répondre à des questions sur divers sujets. Le but principal est de tester les connaissances des joueurs et de récompenser ceux qui répondent correctement aux questions posées.

Ce projet utilise une base de donnée locale elle n'est pas encore mise en ligne ,je m'exuce pour la gêne occasionné
